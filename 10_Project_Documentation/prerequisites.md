# Prerequisites

Before setting up the Credit Card Approval Prediction System, ensure you have the following prerequisites installed and configured:

## 1. Operating System
- **Windows 10/11** (Standard environment used for development and testing).
- Alternatively, macOS (10.15+) or Ubuntu (20.04+).

## 2. Command Line Tools
- **Git**: Essential for cloning the repository and managing versions. Download from [git-scm.com](https://git-scm.com/).
- **PowerShell / Bash / Command Prompt**: Modern terminals to run script commands.

## 3. Package Managers (Choose One)
- **Anaconda** (Recommended): Provides robust environment isolation and pre-compiled mathematical binaries. Install from [anaconda.com](https://www.anaconda.com/).
- **Miniconda**: A lightweight version of Anaconda.
- **Python (Standard)**: Standard installation with `pip` and `venv`. Download from [python.org](https://www.python.org/).

## 4. Integrated Development Environment (IDE)
- **VS Code**: Visual Studio Code editor recommended.
  - Install the **Python** extension by Microsoft.
  - Install the **Jupyter** extension by Microsoft.
