# Hardware Requirements

The hardware requirements for running model training notebooks and serving the Flask web application locally are described below:

## Minimum Specifications
- **CPU**: Dual-core x86_64 Processor (Intel Core i3 or equivalent AMD Ryzen 3, 2.0 GHz+).
- **RAM**: 8 GB DDR4.
- **Storage**: 2 GB of available solid-state/hard drive space (includes dataset files, Anaconda dependencies, and exported model weights).
- **Display**: 1366x768 resolution monitor for data visualizations and browser testing.

## Recommended Specifications (For Fast Training and Inference)
- **CPU**: Quad-core x86_64 Processor (Intel Core i5/i7 or AMD Ryzen 5/7, 2.5 GHz+).
- **RAM**: 16 GB DDR4 or DDR5.
- **Storage**: SSD with at least 5 GB of free space.
- **GPU**: N/A (Standard CPU operations are fully sufficient for dataset dimensions used in this project).
